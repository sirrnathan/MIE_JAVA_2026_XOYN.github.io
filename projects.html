Featured Project

Remote Data Acquisition for Smart Meter

Technologies Used:
HTML5 CSS3 JavaScript Node.js Express.js SQLite EJS Git
Year: 2025
Role: Lead Full-Stack Developer

Overview

xnovest-meter-portal is a web-based meter data management system specifically designed to address operational challenges faced by consulting engineers at Xnovest Africa. The system focuses on streamlining the process of accessing, analyzing, and reporting three-phase electrical consumption data, which forms the foundation for energy efficiency recommendations and system optimization strategies.

Key Features

  • Role-Based Access Control (RBAC): Secure authentication with different access levels for administrators and regular users.
  • Interactive Dashboards: Visual representation of meter data for quick trend analysis.
  • CSV Export Functionality: Export raw data for further analysis in external tools.
  • Data Isolation: Secure protocols ensuring client data confidentiality.
  • Meter Reading Management: Centralized system for viewing and managing three-phase electrical consumption data.

Technical Implementation

The application was built using the Node.js/Express.js stack with SQLite as the database. Server-side rendering was implemented using EJS templates. The system architecture follows MVC pattern for clean separation of concerns. Authentication is handled with session-based auth, and data visualization is implemented with Chart.js for interactive graphs.

Impact

This system enabled engineers at Xnovest to analyze consumption trends 60% faster than previous manual methods. The centralized data management improved team efficiency and provided a reliable foundation for energy efficiency recommendations to clients.
